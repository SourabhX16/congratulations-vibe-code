## Local Development

```bash
npm install

```

## Deploy to GitHub Pages

1. Update `vite.config.js` base path to match your repo name
2. Run:
```bash
npm run deploy
```

3. Enable GitHub Pages in repo settings (use `gh-pages` branch)
